## Welcome to pleasedonate
**pleasedonate** is an platform for helping people who really need help. It simple works with GitHub pages and is easily configurable.
Even though this is an example repository, we have some donations listed.

### Supported languages
Right now, the following languages are supported:
 - German
 - English

... other languages will be maybe be supported in future (feel free to create an PR).

### Other platforms
**pleasedonate** itsself does *NOT* support payments. Donations are links to other platforms like gofundme.com and more.
